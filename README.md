# 语音录制程序(集成Agora)
本程序集成Agora实现了语音录制功能

### 启动方法

1. 安装依赖

```
pip install -r requirements.txt
```

2. 配置环境变量

编辑.env文件，将Agora和阿里云OSS的参数进行配置

3. 运行应用

```
python app.py
```

4. 访问前端

打开浏览器访问 `http://localhost:5000`
